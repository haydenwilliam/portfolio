PlateMath

Powerlifting Assistant App

Back to Projects

Project Overview

PlateMath is a powerlifting training companion app built for iOS. It allows users to input their one-rep maxes for squat, bench, and deadlift, calculating Rate of Perceived Exertion (RPE) coefficients for various sets and reps. The app also displays an interactive barbell animation, helping lifters visualize exactly how to load plates for their working sets.

Key Features

  • One-rep max tracking for the primary powerlifting movements
  • RPE calculation based on weight, sets, and reps
  • Barbell loading visualization with animated plate arrangement
  • Support for both metric (kg) and imperial (lb) weight systems
  • Clean, workout-friendly interface for quick access during training

Technologies Used

Swift SwiftUI iOS Development UI Animation

The application was built using Swift and SwiftUI, Apple's modern declarative UI framework, creating a responsive and visually appealing experience for iOS users.

Implementation Details

The core functionality of PlateMath includes:

  1. Tracking one-rep max values for the major powerlifting movements
  2. Calculating working weights based on percentage of one-rep max
  3. Determining RPE (Rate of Perceived Exertion) values for different weight/rep combinations
  4. Visually representing barbell loading with a clean, animated interface

SwiftUI's declarative approach allowed for creating an interface that's both visually appealing and responsive, with smooth animations for the barbell loading visualization.

Key Learnings

This project provided valuable experience with building iOS applications using SwiftUI, implementing custom animations, and creating a genuinely useful tool for the powerlifting community. User feedback helped refine the interface to be practical during actual workout sessions.
